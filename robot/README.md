## 常见问题
**机器人不说话怎么办？**

 1. 检查群是否开启匿名消息（若机器人关闭了匿名发送消息可忽略此项），若关闭则开启
 2. 检查机器人是否被禁言（普通和匿名），若被禁言则解禁或等待禁言解除
 3. 在机器人官方群发相关指令检查机器人是否回复，若未回复则联系机器人负责人，否则自行处理

**机器人收费吗？**

稳定后将收费，价格待定。
